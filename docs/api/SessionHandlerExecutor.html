Class: SessionHandlerExecutor

SessionHandlerExecutor

new SessionHandlerExecutor(startHandlersopt, endHandlersopt)

The SessionHandlerExecutor manages the execution of the registered SessionHandlers. This uses an async supporting loop to execute each SessionHandler and does not expect a callback or return object.
