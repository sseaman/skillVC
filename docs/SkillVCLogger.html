Class: SkillVCLogger

SkillVCLogger

new SkillVCLogger(levels)

Providers a centralized logger mechanism with per object definable logging levels using winston The levels can be set by providing a map where the keys are the objects and the values the log levels. If no level is provided, it defaults to info If the map contains key 'all' then every log will be set to the level defined with the 'all' key If the level is set to 'none', logging is disabled
Parameters:
Name Type Description
levels Map The logging levels

Example
var Logger = require('./skillVCLogger.js');
var log = Logger.getLogger('SkillVC'); // use to actually do the logging

Then in your code:
Logger.setLevels({'all':'debug'}); // everything set to debug
Logger.setLevels({'all':'none'});  // turn off all logging
Logger.getLevels({'SkillVC' : 'debug', 'FilterManager' : 'info'}) // turns on debugging for SkillVC.  All others will be turned off
